One-time Firebase Setup — takes 3 minutes
1
Go to console.firebase.google.com → Create a project (any name, e.g. windmill-stock) → Disable Google Analytics → Create Project.
2
In your project, click Build → Realtime Database → Create Database. Choose any location. Select "Start in test mode" → Enable.
3
Click the ⚙️ gear icon → Project Settings → scroll to "Your apps" → click </> Web. Register the app (any name). Copy the firebaseConfig values into the fields below.
🔒 Firebase free tier (Spark plan) — 1 GB storage, 10 GB/month. No credit card needed.
After setup, your boss and team see the same live data from any device, anywhere.
